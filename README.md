# LaserSell Brand Kit

Official brand assets for LaserSell. Use these when referencing or integrating with LaserSell in your projects, articles, or applications.

## Assets

| File | Description |
|------|-------------|
| `icon/lasersell.svg` | Icon (SVG) |
| `icon/lasersell.png` | Icon (PNG, 512x512) |
| `logo/lasersell-full-light.png` | Full logo for dark backgrounds |
| `logo/lasersell-full-dark.png` | Full logo for light backgrounds |

## Download

[Download ZIP](https://github.com/lasersell/brand-kit/archive/refs/heads/main.zip)

## Usage Guidelines

- Do not alter, rotate, or distort the logo.
- Maintain clear space around the logo.
- Use the light variant on dark backgrounds and the dark variant on light backgrounds.
- Do not use the LaserSell logo to imply endorsement without permission.
